Утилита сервера точного времени, который "врет" на заданное в
файле configuration.txt количество секунд.

Сервер работает по протоколу UDP на 123 порту.

Чтобы взаимодействовать с утилитой, запустите скрипт server.py -
поднимется сервер.Затем запустите скрипт client-py и в консоли
будете видеть "обманчивое время".

Запуск сервера - python server.py
Запуск клиента - python client.py

Фейковое количество секунд написать в формате числа в файл configuration.txt
